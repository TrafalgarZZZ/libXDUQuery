# libXDUQuery
西电查询服务接口

### 🏆 获取并使用
- Download ZIP
- The latest compiled JAR is in the `dist` directory
- Add the JAR to your project

### 😀 已支持
- **校一卡通 -> ECard**
- **体育打卡 -> SportsClock**
- **物理实验 -> PhysicsExperiment**
- **水电用量 -> WaterAndElectricity**

### 😂 待支持
- **图书馆 -> Library**
