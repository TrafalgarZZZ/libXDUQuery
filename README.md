# libXDUQuery
西电查询服务接口

### 🏆 获取与使用
- Download ZIP
- The latest compiled JAR is in the `dist` directory
- Add JAR to your project

### 😀 已支持
- **校一卡通 -> ECard**
- **体育打卡 -> SportsClock**
- **物理实验 -> PhysicsExperiment**
- **水电用量 -> WaterAndElectricity**

### 😂 待支持
- **校图书馆 -> SchoolLibrary**
- **校网流量 -> CampusNetwork**
